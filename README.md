# Alfa Yazılım - Modern Yazılım Şirketi Websitesi

Alfa Yazılım için tasarlanan modern, responsive ve kullanıcı dostu web sitesi.

## 🚀 Özellikler

- **Modern Tasarım**: Temiz ve profesyonel görünüm
- **Responsive**: Tüm cihazlarda mükemmel görüntü
- **Hızlı ve Optimize**: Optimize edilmiş performans
- **SEO Uyumlu**: Arama motorları için optimize
- **Animasyonlar**: Smooth geçişler ve etkileşimler
- **İletişim Formu**: Functional contact form

## 📱 Responsive Tasarım

Web sitesi aşağıdaki ekran boyutlarında test edilmiştir:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (320px - 767px)

## 🛠 Teknolojiler

- **HTML5**: Semantik yapı
- **CSS3**: Modern styling, Flexbox, Grid
- **JavaScript ES6+**: İnteraktif özellikler
- **Font Awesome**: İkonlar
- **Google Fonts**: Inter font family

## 🎯 Bölümler

1. **Hero Section**: Ana başlık ve CTA butonları
2. **Hizmetler**: Web geliştirme, mobil uygulama, bulut çözümleri, yapay zeka
3. **Hakkımızda**: Şirket bilgileri ve istatistikler

## 🚀 Kurulum

1. Dosyaları web sunucunuza yükleyin
2. `index.html` dosyasını açın
3. Web sitesi kullanıma hazır!
